Lab Exam Practice (HTML Foundations)

These practice questions are within Lab 01 scope. They focus on semantic HTML, navigation structure, accessibility basics, and tables.

Part A: Find Mistakes (Identify + Fix)

In the exam, you will (1) identify the mistake in one sentence and (2) write the corrected HTML.

Part B: True / False

  • T/F: Every link inside <ul> must be inside <li>.
  • T/F: A page may have multiple <h1> headings for styling.
  • T/F: aria-current="page" marks the active navigation item.
  • T/F: A form is accessible without labels.
  • T/F: KLM predicts execution time for expert users.
  • T/F: Semantic elements improve accessibility and meaning.
  • T/F: <div> conveys meaning by itself.
  • T/F: Captions are optional for accessible tables.
  • T/F: Headings should follow logical hierarchy.
  • T/F: Images must always have alt text.

Part C: Short Answer

  1. Explain the difference between a semantic element (e.g., nav) and a generic element (e.g., div).
  2. Why does KLM not model high-level planning and interpretation?
  3. What is the purpose of aria-label in navigation?
  4. Why should headings not skip levels?
  5. How does a caption improve table accessibility?
  6. What is the difference between id and for in form labels?
  7. Why is lang attribute important in HTML?
  8. Explain why alt text is critical for screen readers.
  9. What is the difference between semantic and presentational markup?
  10. Why should only one navigation item have aria-current="page"?
